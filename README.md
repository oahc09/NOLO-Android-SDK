# NOLO-Android-SDK  
You can use this SDK to develop android applications that supports NOLO.

[NOLO-Java-Android-SDK](./NOLOVR/Java)

[NOLO-C-Android-SDK](./NOLOVR/C)

Documentation for the C API is available on [here](./Docs/Docs_for_C).
